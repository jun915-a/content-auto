# GitHub CLIのテレスコープ

GitHub CLIは、ユーザーの行動を追跡するためのテレスコープ機能を導入しました。この機能により、GitHubはユーザーの使い方を把握し、CLIの機能を改善することができます。テレスコープ機能は、ユーザーの個人情報を収集せず、匿名で収集したデータを使用します。

## このトピックの本質
GitHub CLIのテレスコープ機能は、ユーザーの行動を追跡し、データを収集するための機能です。
## 5秒で分かるポイント
- **Point 1**: テレスコープ機能はユーザーの行動を追跡します
- **Point 2**: データは匿名で収集されます
- **Point 3**: ユーザーの個人情報は収集されません
## 詳細解説
**Element 1**: テレスコープ機能は、ユーザーの使い方を把握するために使用されます。
**Element 2**: データはGitHubのサーバーに送信され、分析されます
> ポイント: テレスコープ機能は、ユーザーの個人情報を保護するために設計されています。
## 実世界への影響
- ユーザーの使い方が把握できる
- CLIの機能が改善される
- ユーザーの個人情報が保護される

テレスコープ機能は、ユーザーの使い方を把握し、CLIの機能を改善するための重要なツールです。
